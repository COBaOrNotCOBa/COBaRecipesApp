# COBaRecipesApp

Приложение с рецептами, сделанное в рамках курса по изучению Android. Проект для отработки базовых навыков: работа с экранами, списками, навигацией и структурой Android-приложения.

## Возможности

- Просмотр списка рецептов
- Детальный экран с описанием рецепта
- Простейшая навигация между экранами

## Технологии

- Kotlin
- Android SDK

## Как запустить

1. Открыть проект в Android Studio.
2. Собрать и запустить приложение на эмуляторе или реальном устройстве (min SDK и другие детали — по настройкам проекта).

## Статус проекта

Учебное приложение по курсу Android. Можно дорабатывать и расширять.
