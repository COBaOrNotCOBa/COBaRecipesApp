# COBA Recipes App

Приложение с рецептами, сделанное в рамках курса по изучению Android. Проект для практики архитектуры, навигации, работы с сетью и локальным хранилищем.

## Что умеет приложение

- Показывать список рецептов
- Открывать экран с подробной информацией о рецепте
- Демонстрировать типовой стек для «реального» Android-приложения

(Конкретные экраны и фичи зависят от текущей реализации — при необходимости можно дополнить этот раздел.)

## Технологический стек

В проекте настроен следующий стек библиотек и плагинов:

- **Язык:** Kotlin
- **UI:** AppCompat + Material Components
- **DI:** Dagger Hilt
- **Навигация:** AndroidX Navigation + Safe Args
- **Сеть:**
  - Retrofit
  - OkHttp + Logging Interceptor
  - Kotlinx Serialization (конвертер для Retrofit)
- **Локальное хранилище:** Room (runtime + compiler через KSP)
- **Изображения:** Glide
- **Прочее:**
  - AndroidX Core / Fragment KTX
  - JUnit, AndroidX Test, Espresso для тестов

## Требования и запуск

1. Установить последнюю версию **Android Studio** с поддержкой Kotlin и Gradle.
2. Клонировать репозиторий:
   ```bash
   git clone https://github.com/COBaOrNotCOBa/COBaRecipesApp.git
Открыть проект в Android Studio — Gradle сам подтянет плагины и зависимости.

Собрать и запустить приложение на эмуляторе или реальном устройстве (min SDK и другие параметры см. в app/build.gradle.kts).
